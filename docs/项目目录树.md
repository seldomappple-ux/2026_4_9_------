# 项目目录树

```text
.
|-- 00-公共资源/ (公共上游素材与共享提示词来源)
|   |-- 世界观设定/ (世界观设定、风格边界)
|   |-- 人物信息/ (角色总入口)
|   |   |-- 人物介绍/ (角色设定文字资料)
|   |   |-- 参考图/ (角色灵感图、服装图、发型图)
|   |   `-- 衍生提示词/ (由角色资料整理出的基础提示词)
|   |-- 原始图像/ (原始图像、立绘、平视图)
|   |-- 参考图/ (参考图、姿态图、风格图)
|-- prompts/ (按阶段和方案汇总查看提示词版本)
|   |-- 00-阶段0-原始元素处理/
|   |   |-- 原始图清洗/
|   |   |-- 角色元素提取/
|   |   |-- 风格统一/
|   |   `-- 基础描述整理/
|   |-- 01-第一阶段-线稿/
|   |   |-- 方案01-直接生成/
|   |   |-- 方案02-骨架生成/
|   |   `-- 方案04-换头换衣服生成/
|   |-- 02-第二阶段-上色/
|   |   |-- 方案01-单人上色/
|   |   `-- 方案02-多人上色/
|   `-- 03-第三阶段-动作/
|       `-- 方案01-单人动作/
|-- 01-第一阶段-线稿/ (第一阶段：线稿目标与前置生成方案)
|   |-- 方案01-直接生成/ (直接生成前置图方案)
|   |   |-- 提示词/ (节点提示词)
|   |   |-- 节点截图/ (节点与参数截图)
|   |   |-- 输出结果/
|   |   |   `-- 第01轮/ (第1轮输出)
|   |   `-- 测试记录/ (测试记录与问题总结)
|   |-- 方案02-骨架生成/ (骨架生成方案)
|   `-- 方案04-换头换衣服生成/ (换头换衣服前置方案)
|-- 02-第二阶段-上色/ (第二阶段：上色方案)
|   |-- 方案01-单人上色/ (单人上色方案)
|   `-- 方案02-多人上色/ (多人上色预留方案)
|-- 03-第三阶段-动作/ (第三阶段：动作生成方案)
|   `-- 方案01-单人动作/ (单人动作方案)
|-- docs/ (规范、计划、对比报告)
|   |-- 文件结构定义.md (结构定义源文件)
|   `-- 项目目录树.md (目录示意)
`-- archive/ (废弃方案与历史归档)
```

说明：

- `人物信息/` 用来沉淀每个角色的文字资料、灵感图和衍生提示词
- 根目录 `prompts/` 用来集中查看不同阶段和方案的提示词调整版本
- `prompts/00-阶段0-原始元素处理/` 用来记录原始素材预处理相关提示词
- 每个方案目录内部都包含相同的 `提示词/`、`节点截图/`、`输出结果/第01轮/`、`测试记录/`
- 第一阶段不是单一方案，而是围绕线稿目标的多个可比较方案集合
- 上游素材通过 `00-公共资源/` 汇总，再分发到各方案节点提示词
