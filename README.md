# 图生图节点流对比测试项目

本仓库用于沉淀三阶段图生图测试资料，重点是找出更合适的提示词组合与工作流匹配方式。

## 当前测试主线

- 第一阶段目标是线稿，比较直接生成、骨架生成、换头换衣服生成等方案。
- 第二阶段目标是上色，当前包含单人与多人方案位。
- 第三阶段目标是动作生成，当前先测试单人动作。

## 上游素材链路

所有方案都从公共素材出发：

- 世界观设定
- 人物信息
- 原始素材

这些素材会先经过阶段0的原始元素处理，再进入各阶段的提示词与工作流测试。

## 仓库使用方式

- 公共原始资料放在 `00-公共资源`
- 体裁级标准定义放在 `00-公共资源/体裁定义`
- 角色相关资料统一放在 `00-公共资源/人物信息`
- 全局提示词总表放在根目录 `prompts`
- 各阶段方案放在 `01-第一阶段-线稿`、`02-第二阶段-上色`、`03-第三阶段-动作`
- 每个方案都统一包含 `提示词`、`节点截图`、`输出结果`、`测试记录`
- 长期规则和连续对话记录集中放在 `.agents`
