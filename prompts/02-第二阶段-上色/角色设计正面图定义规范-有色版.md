# 角色设计正面图定义规范（有色生产级版本）

## 核心术语定义

**标准术语**：角色正交彩色参考视图（Character Orthographic Color Reference）/ 生产级标准彩色正视图（Production-Ready Colored Frontal View）

**底层逻辑定位**：角色彩色正交基准规范（Character Colored Blueprint Specification），为原画定稿、3D材质贴图制作、IP视觉标准化、衍生品设计提供唯一色彩与造型基准

---

## 四大核心维度

### 1. 几何投影方式 (Projection)
- **核心词**：`Orthographic Front View`（正交正面视图）
- **定义**：彻底消除透视缩减（No foreshortening），身体远近处的线条与色块在二维平面上具有完全相同的度量意义，无近大远小、无透视变形，确保造型比例可直接用于3D建模对位

### 2. 标准化姿态 (Standardized Pose)
- **核心词**：`A-Pose` 或 `T-Pose`
- **定义**：肢体呈放射状分布，腋下与胯部保持净空；手掌通常朝前或自然下垂，确保关节（Joints）位置的拓扑结构（Topology）、服饰色块分割、配饰完整可见，无肢体遮挡、无造型重叠，适配后续3D骨骼绑定与UV拆分

### 3. 上色与线条技术规格 (Coloring & Line Art Standards)
- **核心词**：`Standardized Flat Coloring` / `Technical Line Art` / `Constant Line Weight` / `Pantone Standard Color Matching`
- **定义**：
  1. 线条规范：使用等粗的封闭黑色轮廓线勾勒，线条边界清晰锐利，无模糊、无断点、无粗细渐变，确保色块分区完全闭合，可直接提取线稿用于3D拓扑
  2. 上色规范：采用二次元工业级标准平涂上色，无渐变、无厚涂、无写实光影、无素描调子、无环境光反射；仅保留最小限度的闭塞阴影（Occlusion）用于区分结构层级，阴影色必须基于主色同色系加深，禁止使用纯黑阴影
  3. 色彩规范：所有色块使用固定色值，主色、辅助色、点缀色分区明确，无溢色、无杂色、无色块溢出轮廓线，色彩体系完全匹配角色设定的色卡规范，确保全渠道视觉统一

### 4. 结构与色彩说明性 (Structural & Color Illustration)
- **核心词**：`Structural Breakdown` / `Clear Color Block Segmentation` / `Clear Silhouette`
- **定义**：
  1. 造型结构：强调角色外轮廓（Silhouette）和内部零件的物理分割（Segmentation），每一道线条都代表一个真实的物理转折或零件缝隙，服饰分层、配饰结构、身体结构清晰可辨
  2. 色彩结构：每一个独立色块对应一个独立的物理结构/材质，同材质同部件使用统一色值，通过色彩区分服饰层级、材质差异、结构转折，为3D材质贴图、衍生品生产提供精准的色彩与分区基准

## 标准提示词结构（Prompt Logic）

```text
(Category): Character Orthographic Color Reference Sheet
(View): Flat Frontal View, 2D Orthographic Projection
(Pose): Standard A-pose, symmetrical standing, no limb occlusion
(Line Style): Clean technical line art, constant line weight, precise closed outline, high contrast, non-sketchy
(Coloring Style): Standardized anime flat coloring, solid color blocks, minimal occlusion shadow, no gradient, no complex shading, no painting texture
(Requirements): No perspective, no foreshortening, no background, solid white background, full body alignment, visible joint structure, clear color block segmentation, accurate color matching, full body shot
```

## 提示词构造器参数说明

### 姿态选择
- **A-Pose**：标准生产姿态，利于肩膀拓扑与服饰结构展示，最常用于工业化角色生产、3D建模对接
- **T-Pose**：骨骼绑定基准姿态，适合3D绑定、动作捕捉对接、服装版型无遮挡展示
- **Natural Front**：自然站立正面姿态，适合角色IP宣传、原画展示、非建模向的视觉定稿

### 上色精度
- **Industrial Standard Flat**：工业级标准平涂，无渐变无复杂光影，色块闭合精准，色值统一，适配3D建模、衍生品生产、全渠道IP标准化，为核心推荐选项
- **Cel Shading**：赛璐璐标准上色，保留规范的层级阴影，无渐变，适合动画原画、IP视觉展示
- **Variable Flat**：艺术化平涂，可保留轻微材质质感表现，适合原画宣传、概念展示，非工业生产首选

### 结构重心
- **Anatomy Focus**：人体解剖重心，强调骨骼与肌肉结构的色彩区分
- **Component Focus**：零件与层级重心，强调服饰、配饰的色块分区与结构拆分，适合复杂服饰、多配饰、机械元素角色
- **Color Focus**：色彩体系重心，强化主色、辅助色、点缀色的规范搭配，适配IP视觉标准化、色卡制定
- **Silhouette Focus**：轮廓剪影重心，强化角色外轮廓的辨识度，适合角色概念定稿、IP标识化设计

### 细节等级
- **High Density**：高密度细节，精准还原服饰纹理、面料质感、配饰精细结构、完整色彩层级
- **Medium Density**：中等密度细节，保留核心结构与关键色彩分区，为常规角色工业生产标准
- **Low Density**：极简结构，仅保留核心造型轮廓与主色色块，适合概念设计初期、快速迭代
