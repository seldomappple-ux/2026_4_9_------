# 前景人物彻底移除+背景无损补全专用提示词

## 核心唯一任务
对输入图像执行**前景人物100%彻底移除+人物遮挡区域背景无缝补全**操作，无任何额外任务、无人物保留、无画面美化修改，最终输出单张「无任何人物及人物相关残留、背景完整无缺失、与原场景视觉逻辑完全统一」的纯净背景图像。

## 执行强制规范

### 一、人物及关联痕迹彻底清除规范
1. 人物本体全量清除：完整移除画面中所有前景人物主体，包括人物全身轮廓、发丝、服饰、配饰、肢体等所有人物相关元素，无任何人物本体的残留、半透明像素、轮廓痕迹。
2. 阴影与边缘残留彻底清除：同步彻底移除人物周边的所有边缘投影、环境落影、地面/墙面的人物阴影、人物边缘的色彩溢出、抠图残留痕迹，无任何与人物相关的光影残留。
3. 原生背景零改动规则：除人物及人物阴影覆盖的区域外，原背景的所有内容100%完整保留，禁止任何形式的修改、美化、重绘、细节增减。

### 二、背景缺失区域补全规范
1. 补全逻辑强一致性：严格遵循原背景的空间透视关系、光影方向、色调风格、纹理材质、场景物理逻辑进行补全，补全内容必须与原背景完全适配，无逻辑违和。
2. 无缝衔接要求：补全区域与原背景的衔接处必须完全无缝，无拼接痕迹、无色彩断层、无光影错位、无纹理突变。
3. 内容合规约束：补全内容仅还原原场景应有的背景内容，禁止添加原场景不存在的元素，保证补全后的背景符合原场景的真实逻辑。

## 刚性约束红线
1. 任务排他性约束：仅执行前景人物及阴影移除、背景缺失补全操作，禁止在画面中保留任何人物、人物相关元素、人物阴影。
2. 原背景保护约束：非人物遮挡的原背景区域必须100%与原图一致，禁止任何形式的改动。
3. 画布一致性约束：输出图像的画布尺寸、分辨率、色彩空间、画幅比例与输入原图完全一致。
4. 无额外内容约束：最终输出画面仅包含完整补全后的纯净背景，禁止添加任何水印、文字、边框、标记等多余内容。

## 输出要求
最终仅输出单张符合上述全部规范的完整背景图像，无其他额外输出内容。
